HTML tags are like keywords which defines that how web browser will format and display the content.

With the help of tags, a web browser can distinguish between an HTML content and a simple content.

HTML tags contain three main parts: opening tag, content and closing tag. But some HTML tags are unclosed tags.

All HTML tags must enclosed within < > these brackets.

Every tag in HTML perform different tasks.

If you have used an open tag <tag>, then you must use a close tag </tag> (except some tags)

All about Meta tags:
The <meta> tag is used to provide such additional information like,
1. Specifying Keywords
2. Document Description
3. Document Revision Date
4. Document Refreshing
5. Page Redirection
6. Setting Cookies
7. Setting Author Name
8. Specify Character Set etc...

This tag is an empty element and so does not have a closing tag but it carries information within its attributes.
